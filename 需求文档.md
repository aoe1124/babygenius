宝宝未来职业潜力测试

1. 项目概述

主标题：宝宝未来职业潜力测试
副标题：测一测你的宝宝，未来潜藏着哪种职业天赋？

项目目标：开发一个面向小红书宝妈用户的趣味测试网页，用户支付后获得链接，通过回答关于宝宝日常行为的问题，生成一份具有个性化、可分享的“未来职业潜力报告”，以提升店铺互动与传播。

核心价值：为宝妈提供有趣的育儿观察视角，生成高传播性的社交内容（海报），吸引潜在用户。

2. 用户画像

核心用户：小红书平台上的宝妈，宝宝年龄为0-6岁。

用户需求：在育儿过程中获得趣味互动、记录宝宝特点、产生可用于社交平台分享的内容。

使用场景：利用碎片时间（5-10分钟）完成测试，并保存或分享结果。

3.6 系统需求

3.6.1 数据处理
- 前端计算：所有答案计分和结果生成在浏览器端完成
- 数据存储：不存储用户答案
- 隐私保护：测试结束后自动清除本地数据

3.6.2 性能要求
- 首屏加载：3秒内完成
- 答题响应：点击后500ms内显示反馈

- Chart.js绘制潜能分布图

6.3.3 PDF报告生成
- 使用html2canvas将结果页转换为图片
- 使用jsPDF生成A4尺寸的PDF文档
- PDF内容包含：宝宝昵称、测试日期、主要潜能、次要潜能、潜能分布图、职业建议
- 生成的PDF自动下载，文件名格式：宝宝潜能测试报告_YYYYMMDD.pdf

// PDF生成示例
async function saveAsPDF() {
    const element = document.querySelector('.result-container');
    const canvas = await html2canvas(element, {
        scale: 2,  // 提高清晰度
        logging: false
    });

    const imgData = canvas.toDataURL('image/png');
    const pdf = new jsPDF('p', 'mm', 'a4');

    const imgWidth = 210;  // A4宽度
    const pageHeight = 297;  // A4高度
    const imgHeight = canvas.height * imgWidth / canvas.width;

    pdf.addImage(imgData, 'PNG', 0, 0, imgWidth, imgHeight);

    // 生成文件名
    const date = new Date().toISOString().slice(0, 10).replace(/-/g, '');
    const babyName = document.querySelector('.baby-name').textContent;
    pdf.save(`宝宝潜能测试报告_${babyName}_${date}.pdf`);
}

// 计分算法示例
function calculateScore(answers) {
    let scores = {
        artist: 0,
        musician: 0,
        explorer: 0,
        athlete: 0,
        linguist: 0,
        social: 0,
        logician: 0,
        naturalist: 0
    };

    answers.forEach(answer => {
        // 根据题库配置累加分数
        scores[answer.type] += answer.score;
    });

    // 计算百分比，排序
    return normalizeScores(scores);
